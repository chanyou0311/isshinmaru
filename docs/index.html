一心丸って？

一心丸とは、１月１４日の成人の日に呉のれんが通りに、新成人と新成人を祝いたい人が集う特設ブースイベントと、その企画運営チーム（呉のあにき・あねき会）のことです。

鏡開きやお餅つきをはじめ、趣向をこらした”仕掛け”を用意しています。新成人に「また呉に戻ってきたい！」「呉って素敵なまちじゃね！」って、思ってもらえるような時間を演出し倒しています！
